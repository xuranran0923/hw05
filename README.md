# hw05
# 计算机视觉作业：基于CNN的MNIST手写数字识别
## 一、作业概述
本作业基于PyTorch框架，实现两种卷积神经网络（极简CNN、LeNet-5），完成MNIST手写数字识别任务，对比两个模型的训练效果与识别性能，同时记录调试过程与实验结论，完成完整的实验报告。

## 二、环境说明
- 操作系统：Windows 10
- Python版本：3.14.2（请替换为自己实际的Python版本）
- 深度学习框架：PyTorch 2.11.0+cpu
- 依赖库：详见 requirements.txt

## 三、目录结构
PythonProject/
├─ simple_cnn.py # 任务一：极简 CNN 模型代码（实现基础识别功能）
├─ lenet5.py # 任务二：LeNet-5 模型代码（经典网络结构）
├─ requirements.txt # 依赖库清单（一键安装所有所需依赖）
├─ report.md # 实验报告（含模型对比、结构说明）
├─ debug_notes.md # 调试记录（真实报错与解决方法）
└─ data/ # 自动下载的 MNIST 数据集（运行代码后生成）
